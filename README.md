# Rick and Morty Universe

React + TypeScript app displaying Rick & Morty characters and episodes with search, details, and loading states.

## Features
- Search characters by name
- View character details including episodes
- Episode data fetched efficiently with TanStack Query (React Query) and Axios
- Dynamic routing with React Router v6
- Loading states, error handling, and "not found" messages implemented
- Styled with Tailwind CSS for a modern, responsive UI

## Installation
```bash
npm install
npm run dev
